# 🤖 Smart OCR App

A modern AI-powered OCR (Optical Character Recognition) web application built using EasyOCR and Streamlit.

---

## 🚀 Features

* 📤 Drag & Drop Image Upload
* 🔍 AI-based Text Detection
* 📦 Bounding Box Highlighting
* 🌍 Multi-language Support (English, Hindi)
* 📥 Download Extracted Text (TXT & JSON)
* ⚡ Fast and Interactive UI

---

## 🛠 Tech Stack

* Python
* Streamlit
* EasyOCR
* OpenCV
* NumPy


## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📌 How It Works

1. Upload an image
2. EasyOCR processes the image
3. Detects and extracts text
4. Displays results with bounding boxes
5. Download extracted text
